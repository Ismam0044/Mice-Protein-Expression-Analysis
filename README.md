What this project does

Cleans and preprocesses the raw protein data to handle missing values and normalize measurements
Visualizes protein distributions and checks how proteins relate to each other
Applies machine learning models to classify mice into their respective groups based on protein levels
Uses PCA (Principal Component Analysis) to reduce the data dimensions and uncover hidden patterns

What we found
Some proteins turned out to be much better than others at telling the groups apart. 
These proteins could potentially serve as biological markers for neurological conditions like Down syndrome. 
PCA also revealed that the mice naturally form meaningful clusters, which aligns with their biological differences.
